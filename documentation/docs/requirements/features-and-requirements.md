# Features and Requirements

## Functional Requirements
- Chatbot will send a message once a day that encourages team engagement and communication
- Chatbot will keep track of how many times each user responds
- Chatbot will be able to display a leaderboard showing who has interacted with it the most

## Nonfunctional Requirements
- Leaderboard will give users an incentive to communicate with each other
- Chatbot will be available on multiple different grpup discussion platforms (Slack, Discord, SMS, etc.)
